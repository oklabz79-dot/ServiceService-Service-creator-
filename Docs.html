ServiceService Module API

A utility module for registering and retrieving custom and engine services.

⚙️ Functions

ServiceService:Create(name, functionTable, propsTable)

Registers a new custom service with its functions and properties.

ParameterTypeDescription
namestringThe unique name for the service.
functionTabletableA table of functions the service will expose.
propsTabletable?Optional initial properties for the service.

Returns: Prints service summary to console (no value returned in Lua).

ServiceService:GetCreatedService(name)

Retrieves a custom service previously registered via :Create().

ParameterTypeDescription
namestringThe name of the custom service to retrieve.

Returns: table | nil - The service object or nil if not found.

ServiceService:GetService(name)

Retrieves a custom service or a standard Roblox engine service (e.g., "Players").

ParameterTypeDescription
namestringThe name of the service (custom or engine).

Returns: table | Instance | nil - The requested service object.
